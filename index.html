Quick Add via URL Parameter

Create countdowns instantly by adding parameters to the URL. If the title is missing, you'll be prompted to enter one!

  • ?time=10mins (Supports m, h, d)
  • ?date=25dec (Smart auto-year logic)
  • ?date=2026-01-01&title=NewYear

Security Passcode
